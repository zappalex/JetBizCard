# JetBizCard
Single view app that is a very basic intro to compose, including a main "header" type section and a collapsible list view. 

## This App Covers: 
- Surface
- Card
- Column
- Row
- Image
- Box
- Button + Remember
- Lazy Column ( RecylerView )
- Modifiers
- Shapes
- Vertical + Horizontal Arrangement
